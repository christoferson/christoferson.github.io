Java Module Notes

  1. Module may not have classes with unnamed package (i.e. those which belong to "default" package)

Module Types

  1. System Modules - Includes Java SE (java.) and JDK (jdk.) modules
  2. Application Modules - Named and defined in the module-info.java
  3. Automatic Modules - Plain jars in the module path. Automatic-Module-Name header in its manifest (META-INF/MANIFEST.MF), it defines the module's name, otherwise the JAR file name is used to determine the name.
  4. Unnamed Module - JAR is loaded onto the classpath
