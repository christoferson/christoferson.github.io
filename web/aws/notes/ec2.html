Price Class

  1. On-Demand instances, you pay for compute capacity by the hour or the second depending on which instances you run.
  2. Spot instances allow you to request spare Amazon EC2 computing capacity for up to 90% off the On-Demand price.
  3. Savings Plans are a flexible pricing model that offer low prices on EC2 and Fargate usage, in exchange for a commitment to a consistent amount of usage (measured in $/hour) for a 1 or 3 year term.
  4. A Dedicated Host is a physical EC2 server dedicated for your use. Dedicated Hosts can help you reduce costs by allowing you to use your existing server-bound software licenses, including Windows Server, SQL Server, and SUSE Linux Enterprise Server (subject to your license terms), and can also help you meet compliance requirements.
