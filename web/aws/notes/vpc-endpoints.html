AWS VPC Endpoints Notes

  1. Keep traffic within the AWS Network
  2. Private Connection between VPC and AWS Services or Marketplace Partner Services or Endpoint Services in Other Accounts.
  3. Eliminates the need to traverse the Internet. Instances do not require public IP. Redundant and HA component.

Endpoint Types

  1. Gateway Endpoint -> DynamoDB & S3. Target for a specific route in Route Table. Free.
  2. Interface Endpoint -> Other AWS services. PrivateLink Technology. Elastic Network Interfaces (ENI) with a Private IP Address. Cost Money.
