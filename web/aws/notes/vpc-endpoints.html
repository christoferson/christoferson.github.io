AWS VPC Endpoints Notes

PrivateLink is a highly available, scalable technology that enables you to privately connect your VPC to supported AWS services, services hosted by other AWS accounts (VPC endpoint services), and supported AWS Marketplace partner services. You do not need to use an internet gateway, NAT device, public IP address, AWS Direct Connect connection, or AWS Site-to-Site VPN connection to communicate with the service. Traffic between your VPC and the service does not leave the Amazon network and is not exposed to the public internet.
  1. Keep traffic within the AWS Network
  2. Private Connection between VPC and AWS Services or Marketplace Partner Services or Endpoint Services in Other Accounts.
  3. Eliminates the need to traverse the Internet. Instances do not require public IP. Redundant and HA component.
  4. Virtual Device - horizontally scaled, redundant, and highly available VPC components
  5. Private DNS - (Optional, AWS services and AWS Marketplace Partner services only) Enable private DNS for the endpoint so you can make requests to the service using its default DNS hostname. Private DNS is not supported for Amazon S3 interface endpoints.
  6. VPC endpoint — The entry point in your VPC that enables you to connect privately to a service.
  7. Endpoint service — Your own application or service in your VPC. Other AWS principals can create an endpoint from their VPC to your endpoint service.

VPC Endpoint Types

  1. Gateway Endpoint -> DynamoDB & S3. Target for a specific route in Route Table. Free.
  2. Interface Endpoint -> Other AWS services. PrivateLink Technology. Elastic Network Interfaces (ENI) with a Private IP Address. Cost Money. Require you to choose Subnet, Security Group during creation. Only need 1 Endpoint per AZ.
  3. Gateway Load Balancer endpoint -> provides private connectivity between virtual appliances in the service provider VPC and application servers in the service consumer VPC. serves as an entry point to intercept traffic and route it to a service that you've configured using Gateway Load Balancers, for example, for security inspection.
