AWS EFS Notes

Security

  1. Security Group - You control which EC2 instances can access your EFS file system by using VPC security group rules and AWS Identity and Access Management (IAM) policies. Use VPC security groups to control the network traffic to and from your file system.
  2. IAM Policy - Attach an IAM policy to your file system to control which clients can mount your file system and with what permissions
  3. Access Point - EFS Access Points to manage application access
  4. Control access to files and directories with POSIX-compliant user and group-level permissions.
