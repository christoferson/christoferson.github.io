AWS-Specific Considerations:

  • Service Limits and Quotas: Technical constraints
    • CloudWatch Limitations:
      • Metrics:
        • Custom Metrics:
          • Maximum 10 dimensions per metric
          • Resolution: 1 second to 1 hour
          • Data retention: 15 months
        • API Limits:
          • PutMetricData: 150 TPS
          • GetMetricData: 50 TPS
          • Maximum 500 metrics per request
      • Logs:
        • Log Groups:
          • Maximum events size: 256 KB
          • Batch size: Up to 1MB
          • Query timeout: 15 minutes
    • VPC Flow Logs:
      • Configuration Limits:
        • Maximum aggregation interval: 10 minutes
        • Maximum custom fields: 21
        • Log record format size: Up to 255 characters
      • Destination Constraints:
        • S3:
          • Object size limits
          • Partition limitations
          • Encryption requirements
        • CloudWatch Logs:
          • Log stream limits
          • Retention period options
          • Query performance constraints
    • Traffic Mirroring:
      • Session Limits:
        • Maximum sessions per ENI: 3
        • Filter rules per session: 1000
        • Bandwidth: Up to interface capacity
