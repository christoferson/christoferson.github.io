Knowledge 1: Connectivity Options and Their Use Cases

  • VPC Peering vs Transit Gateway:
    • VPC Peering:
      • Direct one-to-one connectivity
      • Lower latency
      • No bandwidth charges between AZs
      • Non-transitive routing
    • Transit Gateway:
      • Hub-and-spoke topology
      • Centralized management
      • Transitive routing
      • Multi-account support

Knowledge 3: Network Interface Types

  • Interface Options:
    • Elastic Network Interface (ENI):
      • Basic networking
      • Multiple IP addresses
      • Security group association
    • Elastic Network Adapter (ENA):
      • Enhanced networking
      • Higher bandwidth
      • Lower latency
    • Elastic Fabric Adapter (EFA):
      • HPC workloads
      • OS-bypass capability
      • Low latency
